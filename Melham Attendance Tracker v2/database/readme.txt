installation guide for the website

LOCALHOST/XAMPP
1. start the Apache and MySQL
2. open a browser then paste the link: http://localhost/phpmyadmin/
3. create a database similar to the file name of the SQL File
4. import the the SQL file to the newly created database
5. go to the api/db folder and follow the api configuration guide

HOSTING SITE/LIVE SERVER
1. go to the link and log in to the specific hosting site
2. go to the MySQL Databases or phpMyAdmin
3. click create database
4. after creating the database copy the given Mysql DB Name, MySQL User Name, MySQL Password, and MySQL Host Name
5. go to the api/db folder of the website and follow the api configuration guide
