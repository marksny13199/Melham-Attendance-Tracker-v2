SMTP GUIDE

HOW TO SETUP SMTP

*Note:  Be sure to follow the STEPS before you add your SMTP Gmail

SMTP GMAIL THAT CURRENTLY USED

OLD SMTP GMAIL THAT WERE USED

ID SMTP GMAIL HIGHLIGHTED RANDOM LETTERS

STEP 1:  Sign up for a free Gmail account to be used in SMTP



STEP 2:  Enable Two Factor Authentication on the google account: Link Here

             -->  Input the correct email and password of the email to be used in SMTP
             -->  Setup Two Factor Authentication
             -->  Click “continue” on the bottom-right side
             -->  Type the phone number to be used in authenticating the Gmail account
             -->  Input the correct google verification code and click “next” on the bottom-right side
             -->  After successfully inputted the correct verification code, you will see this blue badge.
             -->  Click the “turn on” button on the bottom-right side
             -->  Now the two-factor authentication for the Gmail account has been successfully turn on.

STEP 3:  Generate an App Specific Password for the account: Link Here

             -->  Input your correct email and password that will be used in SMTP.
             -->  Click the drop-down button “Select app” and choose “Mail”
             -->  Click the “Select device” and choose what device you are using.In my case, I used windows 10 computer
             -->  Click the “Generate” button on the bottom-right side
             -->  Please copy & save the highlighted random letters because we will need them on the next step.
